## darkme

#### 使用指南

---

###### 用户功能

- 注册

- 登录

###### 文件功能

- 在未登录的情况下，可以作为文件中转站，文件上传后会得到一个四位数的编码，任何拥有此编号的用户可以在网站上方输入框中输入编号下载文件，有效时间为1天；

- 登录后上传文件将会长期存储，可以作为网盘使用，同样可以作为文件中转（带四位可下载编号）；

###### 便签

> 登录后使用

- 便签创建

  >  便签包含标题和正文，至少有一个不为空

  1. 标签包含两种类型，一般和紧急，以颜色区分

  2. 便签包含两种状态，完成和未完成

- 便签状态修改

  > 未完成与完成状态之间的切换

- 便签的删除



###### 小游戏的破解

- 一笔画游戏的自动路径寻找

  > [点我前往](htttps://darkme.cn/stroke)
